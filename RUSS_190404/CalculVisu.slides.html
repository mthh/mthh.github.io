Calcul scientifique et visualisation de données¶

Numpy¶

Numpy va permettre de crer et manipuler des tableaux typs de n dimensions de manire performante.

Les bibliothques utilises par la suite se basent sur des objets de ce type (les colonnes de valeurs dans pandas, valeurs de chaque cellule d'un raster avec rasterio, etc.) ou prennent ces objets en entres (scikit-learn, etc.).

from mpl_toolkits.mplot3d import Axes3D

def randrange(n, vmin, vmax):
    return (vmax - vmin)*np.random.rand(n) + vmin

fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
n = 100

for c, m, zlow, zhigh in [('r', 'o', -50, -25), ('b', '^', -30, -5)]:
    xs = randrange(n, 23, 32)
    ys = randrange(n, 0, 100)
    zs = randrange(n, zlow, zhigh)
    ax.scatter(xs, ys, zs, c=c, marker=m)

ax.set_xlabel('X Label')
ax.set_ylabel('Y Label')
ax.set_zlabel('Z Label')
plt.show()

Sans rentrer dans les dtails de l'API de matplotlib :

  • API riche, documentation complte, nombreux examples, personnalisable, etc.
  • les diffrentes visualisation de cette prsentation (graphiques avec pandas et seaborn, "cartes" avec geopandas) utilisent matplotlib quand ce n'est pas spcifi.
  • d'autres solutions existent (notamment Bokeh et Plot.ly avec sa syntaxe dclarative et sa possibilit de faire facilement des graphiques interactifs).

# On gnre les donnes, et les bins :
data = np.random.uniform(0, 1, 1000)
hhist, hedges = np.histogram(data, bins=20)

# Cration d'une figure en choisissant les dimensions des axes 
ph = figure(toolbar_location=None,x_range=(0, 1), y_range=(0, 100), plot_height=300)

# Personnalitation rapide
ph.xgrid.grid_line_color = None
ph.yaxis.major_label_orientation = np.pi/4

# Ajout des rectangles
ph.quad(bottom=0, left=hedges[:-1], right=hedges[1:], top=hhist, color="purple", line_color="white")

show(ph)
